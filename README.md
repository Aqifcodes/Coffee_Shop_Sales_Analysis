# Coffee Shop Sales Analysis (Excel Dashboard)
An end-to-end Excel analysis of coffee shop sales data, covering data aggregation, pivot-based analysis, and dashboard visualization for sales performance evaluation.

### Dataset Overview
The dataset includes transactional coffee shop sales data with the following key attributes:
- Date and time of sales
- Store location
- Product category and product type
- Product size
- Quantity sold
- Revenue generated

### Tools & Techniques
- Microsoft Excel
- Pivot Tables for aggregation and analysis
- Pivot Charts for visualization
- Conditional Formatting for pattern detection
- Dashboard design for business reporting

### Analysis Performed
- Time-based analysis (daily, monthly, hourly sales trends)
- Location-wise revenue comparison
- Product category and product size contribution analysis
- Identification of peak sales periods

### Key Insights
- Certain store locations consistently outperform others in total revenue.
- Sales peak during specific hours, indicating strong demand during limited time windows.
- A small number of product categories contribute a majority of total revenue.
- Product size significantly impacts revenue contribution across all categories.
- Monthly sales show a clear trend indicating seasonal or demand-based variation.
